# AIShan
Репозиторий проекта хакатона от AI Talent Hub. В качестве работы был выбран кейс Ашана. Планируется создание чатбота-помощника для клиентов, который бы за короткое время мог решить большинство вопросов, которые можно автоматизировать (например оформление возврата товаров). В случае невозможности помочь клиенту, чатбот будет переводить последнего на свободного сотрудника - в исходном варианте вместо сотрудника используется Chat GPT 3.5 turbo.
